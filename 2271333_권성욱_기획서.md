## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 게임명 | 소방관 시뮬레이터 (Firefighter Simulator) |

| 플랫폼 | PC (Windows) |
| 조작 | 키보드(WASD) + 마우스 |

### 게임 설명

플레이어는 소방관이 되어 건물 화재 현장에서 불을 진압하고 임무를 완수한다.  
화재 구역에 진입해 물을 뿌려 불을 끄면 점수가 올라가며, 제한 시간 안에 모든 화재를 진압하면 게임 클리어.

**목표**: 제한 시간(90초) 안에 맵 전체의 화재를 모두 진압한다.

## 2. 필수 기능 6가지 구현 내용

---

### ① Transform

**활용 목적**: 플레이어 이동 및 오브젝트 위치 제어

**구현 방법**:
- `PlayerController.cs` 스크립트에서 WASD 입력을 받아 `transform.Translate()`로 플레이어 이동 구현
- 물 뿌리기 방향을 `transform.forward` 벡터로 결정
- 화재 오브젝트는 Hierarchy에서 직접 배치(코드 불필요)

**핵심 코드**:
```csharp
// PlayerController.cs
void Update()
{
    float h = Input.GetAxis("Horizontal");
    float v = Input.GetAxis("Vertical");
    Vector3 move = new Vector3(h, 0, v);
    transform.Translate(move * moveSpeed * Time.deltaTime, Space.World);
}
```

**에디터 설정**: Player GameObject의 Position / Rotation은 Inspector에서 설정

---

### ② Rigidbody

**활용 목적**: 플레이어에 물리 기반 중력 및 충돌 처리 적용

**구현 방법**:
- Player GameObject에 Rigidbody 컴포넌트를 **에디터(Inspector)에서 추가**
- 코드에서 `GetComponent<Rigidbody>()`로 참조하여 속도 제어
- `Freeze Rotation X, Z` 설정으로 넘어짐 방지 → **에디터에서 설정**

**핵심 코드**:
```csharp
// PlayerController.cs
private Rigidbody rb;

void Start()
{
    rb = GetComponent<Rigidbody>();
}

void FixedUpdate()
{
    float h = Input.GetAxis("Horizontal");
    float v = Input.GetAxis("Vertical");
    Vector3 move = new Vector3(h, 0, v) * moveSpeed;
    rb.velocity = new Vector3(move.x, rb.velocity.y, move.z);
}
```

**에디터 설정**:
- Add Component → Physics → Rigidbody
- Constraints: Freeze Rotation X, Z 체크

---

### ③ Collider

**활용 목적**: 벽 충돌 처리 및 화재 구역 감지 (Trigger)

**구현 방법**:
- 벽·바닥에 `Box Collider` 추가 → **에디터에서 추가**
- 화재 오브젝트에 `Sphere Collider (Is Trigger: true)` 추가 → **에디터에서 설정**
- `OnTriggerEnter`로 화재 구역 진입 시 플레이어 HP 감소 처리

**핵심 코드**:
```csharp
// FireZone.cs (화재 오브젝트에 부착)
void OnTriggerEnter(Collider other)
{
    if (other.CompareTag("Player"))
    {
        other.GetComponent<PlayerHealth>().TakeDamage(10f);
    }
}

void OnTriggerStay(Collider other)
{
    if (other.CompareTag("Player"))
    {
        other.GetComponent<PlayerHealth>().TakeDamage(5f * Time.deltaTime);
    }
}
```

**에디터 설정**:
- 화재 오브젝트 → Add Component → Physics → Sphere Collider → Is Trigger 체크
- 벽 오브젝트 → Add Component → Physics → Box Collider

---

### ④ Prefab

**활용 목적**: 화재 오브젝트와 물 파티클을 재사용 가능한 단위로 관리

**구현 방법**:
- `FirePrefab`: 불꽃 파티클 + 빛(Light) + FireZone.cs + Sphere Collider → **Project 창에서 Prefab 생성**
- `WaterPrefab`: 물 파티클 시스템 + 화재 소화 로직 → **Project 창에서 Prefab 생성**
- 스크립트에서 `[SerializeField]`로 프리팹 참조 후 사용

**핵심 코드**:
```csharp
// FireManager.cs
[SerializeField] private GameObject firePrefab;
[SerializeField] private Transform[] fireSpawnPoints;

void Start()
{
    // 지정된 위치에 화재 생성 (Instantiate는 ⑤번 항목에서 처리)
    foreach (Transform point in fireSpawnPoints)
    {
        Instantiate(firePrefab, point.position, point.rotation);
    }
}
```

**에디터 설정**:
- Hierarchy에서 완성된 오브젝트를 Project 창으로 드래그 → Prefab 생성
- FireManager Inspector에서 firePrefab 슬롯에 프리팹 드래그 연결

---

### ⑤ Instantiate

**활용 목적**: 물 뿌리기 시 물 파티클을 런타임에 동적으로 생성

**구현 방법**:
- 마우스 왼쪽 버튼 클릭 시 플레이어 앞 방향으로 `WaterPrefab` 생성
- 생성된 물 오브젝트는 일정 시간 후 `Destroy()`로 제거
- 물이 화재 Collider에 닿으면 화재 소화 처리

**핵심 코드**:
```csharp
// WaterShooter.cs (Player에 부착)
[SerializeField] private GameObject waterPrefab;
[SerializeField] private Transform firePoint;
[SerializeField] private float waterLifetime = 2f;

void Update()
{
    if (Input.GetMouseButton(0))
    {
        GameObject water = Instantiate(waterPrefab, firePoint.position, firePoint.rotation);
        Destroy(water, waterLifetime);
    }
}
```

```csharp
// Water.cs (WaterPrefab에 부착)
void OnTriggerEnter(Collider other)
{
    if (other.CompareTag("Fire"))
    {
        Destroy(other.gameObject);  // 화재 소화
    }
}
```

**에디터 설정**:
- Player 하위에 FirePoint(Empty GameObject) 생성 → 물 발사 위치 지정
- WaterShooter Inspector에서 waterPrefab, firePoint 연결

---

### ⑥ Animation

**활용 목적**: 플레이어의 이동/정지/물뿌리기 상태에 맞는 애니메이션 재생

**구현 방법**:
- Player에 `Animator` 컴포넌트 추가 → **에디터에서 추가**
- Animator Controller 생성 및 상태 전환 설정 → **Animation 창에서 설정**
- 코드에서 이동 속도와 행동에 따라 파라미터 전달

**Animator 파라미터**:
| 파라미터 이름 | 타입 | 용도 |
|---|---|---|
| MoveSpeed | Float | 이동 속도 (0: 정지, 양수: 이동) |
| IsExtinguishing | Bool | 물 뿌리는 중 여부 |

**핵심 코드**:
```csharp
// PlayerAnimation.cs (Player에 부착)
private Animator animator;

void Start()
{
    animator = GetComponent<Animator>();
}

void Update()
{
    float h = Input.GetAxis("Horizontal");
    float v = Input.GetAxis("Vertical");
    float speed = new Vector3(h, 0, v).magnitude;

    animator.SetFloat("MoveSpeed", speed);
    animator.SetBool("IsExtinguishing", Input.GetMouseButton(0));
}
```

**에디터 설정**:
- Add Component → Miscellaneous → Animator
- Project 창 우클릭 → Create → Animator Controller 생성
- Animator 창에서 Idle, Walk, Extinguish 상태 추가 및 전환 조건 설정

#### Hierarchy 구조

```

├── Main Camera
├── Directional Light
├── Canvas
│   ├── Background (Image)
│   ├── Title (Text - "소방관 시뮬레이터")
│   ├── StartButton       → OnStartGame()
│   ├── SettingsButton    → OnOpenSettings()
│   ├── ControlsButton    → OnOpenControls()
│   ├── QuitButton        → OnQuitGame()
│   ├── SettingsPanel (기본 비활성)
│   │   ├── BGMSlider
│   │   ├── SFXSlider
│   │   └── CloseButton   → OnCloseSettings()
│   └── ControlsPanel (기본 비활성)
│       ├── ControlsImage (조작 안내 이미지)
│       └── CloseButton   → OnCloseControls()
```

#### 기능 목록

| 버튼 | 동작 |
|------|------|
| 시작 | SampleScene 로드 |
| 설정 | BGM / SFX 볼륨 슬라이더 패널 표시 |
| 조작법 | 키 안내 패널 표시 |
| 종료 | Application.Quit() |

#### 에디터 설정

- Build Settings에 MainMenu(0), SampleScene(1) 순서로 추가

- MainMenuManager 스크립트를 빈 오브젝트에 부착 후 Inspector에서 각 슬롯 연결
