# 플레이어 조작 정의서

## 조작 키

| 키 | 동작 | 상태 |
|----|------|------|
| W / A / S / D | 이동 (앞/왼/뒤/오른) | 구현 완료 |
| Left Shift | 이동속도 1.5배 (스프린트) | 구현 완료 |
| 마우스 좌클릭 | 물 발사 | 구현 완료 |
| F | 물체 상호작용 | **미구현 (TODO)** |
| E | 아이템창 열기/닫기 | **미구현 (TODO)** |
| 1 / 2 / 3 / 4 | 퀵슬롯 아이템 사용 | **미구현 (TODO)** |

## 수치

| 항목 | 값 |
|------|----|
| 기본 이동속도 | 5f |
| 스프린트 배율 | 1.5x |
| 회전 속도 | 10f |
| 물 발사 간격 | 0.1초 |
| 물 오브젝트 수명 | 1.5초 |
